##项目描述
此项目用于快速搭建javaee项目；目前本项目运行环境为springboot、mybatis、jdk1.8；
后期打算加入本人这些年工作经历中的一些经常使用的组件及相关组件的简单封装，目的是为了能够快速进入业务实现阶段，减少框架研发过程；
##运行环境(runtime environment) 
`JDK1.8`、`springboot`、`mybatis`；
相关版本号请查看pom文件；
